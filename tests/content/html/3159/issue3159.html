Issue 3159: Show numeric character reference of MathML

x = − b ± b 2 − 4 ⁢ a ⁢ c 2 ⁢ a

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel
  3. Inspect the first minus of the formula above
  4. Click the item Show Entities As Symbols from the HTML panel options menu
    ⇒ The minus should be displayed as −.
  5. Click the item Show Entities As Names from the HTML panel options menu
    ⇒ The minus should be displayed as &minus;.
  6. Click the item Show Entities As Unicode from the HTML panel options menu
    ⇒ The minus should be displayed as &#8722;.

Expected result

  • There should be three different options to display special characters to display them as symbols, using their names or in Unicode format.
