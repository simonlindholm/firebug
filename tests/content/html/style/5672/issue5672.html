Issue 5672: Missing color and image context menu items

This test also covers issue 5677

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Inspect the blue <div> with the Firebug logo above
  4. Inside the Style side panel right-click on the color value (#8c8cff) of the background property inside the #element rule
    ⇒ There should be an option Copy Color.
  5. Click the Copy Color menu item
  6. Paste the clipboard content into a text editor
    ⇒ The copied text should be #8c8cff.
  7. Right-click on the image value (url(firebug.png)) of the background property
    ⇒ There should be an option Copy Image Location.
  8. Click the Copy Image Location menu item
  9. Paste the clipboard content into a text editor
    ⇒ The copied text should be the URL of the image ending with html/style/5672/firebug.png.
  10. Right-click on the image value (url(firebug.png)) of the background property
    ⇒ There should be an option Open Image In New Tab.
  11. Click the Open Image In New Tab menu item
    ⇒ A new browser tab should open showing the Firebug logo.

Expected result

  • There should be special context menu options for colors and images defined in CSS.
