Issue 6817: Copy as cURL: fix multipart request

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Net panel
  3. Click the Send request button above
    ⇒ A request to issue6817.php should be shown
  4. Right-click the request and choose Copy as cURL from the context menu
  5. Paste the text from the clipboard into a text editor

Expected result

  • The clipboard data should contain a cURL command with the -data-binary parameter. The parameter should contain the multipart/form-data payload without binary data.
