Issue 6616: Copy request as a cURL command

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Net panel
  3. Click the Send request button above
    ⇒ A request to issue6616.php should be shown
  4. Right-click the request and choose Copy as cURL from the context menu
  5. Paste the text from the clipboard into a text editor
    ⇒ The clipboard data should contain a valid cURL command.
  6. Set the preference extensions.firebug.net.curlAddCompressedArgument to true
  7. Repeat steps 4 and 5 ⇒ The cURL command should contain the --compressed argument.

Expected result

  • A valid cURL command should be copied to the clipboard.
