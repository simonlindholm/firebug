Issue 5412: Auto-completion for CSS selectors

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Select issue5412.html from the Location Menu
    ⇒ One CSS rule should be listed
  4. Open the rule for editing, and enter various things to check completions:
    • "#" ⇒ "#content"
    • "." ⇒ ".a"
    • "s" ⇒ "span"
    • "#e" ⇒ "#element1"
    • "#element1." ⇒ "#element1.element1"
    • "#element1#" ⇒ "#element1#"
    • ".element1." ⇒ ".element1."
    • "section > h" ⇒ "section > h3"
    • "section>h" ⇒ "section>h3"
    • "#element1 .a." ⇒ "#element1 .a.second"
    • "span, #element1 .a." ⇒ "span, #element1 .a.second"
    • ".a[" ⇒ ".a["
    • ".a[t" ⇒ ".a[t"
    • ".a[data-test=test]." ⇒ ".a[data-test=test].second"
    • ":" ⇒ ":hover"
    • "a::a" ⇒ "a::after"
    • "a::after:h" ⇒ "a::after:h"
    • "a:first-child::a" ⇒ "a:first-child::after"
    • "invalid]syntax:h" ⇒ "invalid]syntax:h"
    • ".second + ." ⇒ ".second + .b"
    • ".second ~ ." ⇒ ".second ~ .b"
    • ".second + #" ⇒ ".second + #"
    • ".second ~ #" ⇒ ".second ~ #"
    • ".second #" ⇒ ".second #"
    • ".second > #con" ⇒ ".second > #con"
    • "* #con" ⇒ "* #content"
    • "* > #con" ⇒ "* > #content"
    • "#frameel" ⇒ "#frameel"
  5. Stop editing, and switch to frame.html in the Location Menu
    ⇒ One CSS rule should be listed
  6. Open the rule for editing, and enter various things to check completions:
    • "#frameel" ⇒ "#frameelement"

Expected result

  • There should be (context-sensitive) auto-completion for CSS selectors.
  • Results should be limited to the frame of the stylesheet.
