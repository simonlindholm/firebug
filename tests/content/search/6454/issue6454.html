Note: This test covers the general HTML panel search as well as issue 7259 and issue 4491.

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel
  3. Focus the search field
  4. Ensure the search option Case Sensitive is unchecked
  5. Search for 'testing'
    There should be 14 occurrences for 'testing' inside the HTML panel.
  6. Check the search option Case Sensitive
  7. Search for 'testing'
    There should be 7 occurrences for 'testing' inside the HTML panel.
  8. Uncheck the search option Case Sensitive
  9. Search for 'Testing'
    There should be 8 occurrences for 'Testing' inside the HTML panel.
  10. Search for '#test div'
    There should be 5 occurrences for '#test div' inside the HTML panel.
  11. Check the search option Case Sensitive
  12. Search for '/html/body'
    There should be 4 occurrences for '/html/body' inside the HTML panel.

Expected results

  • There should always be the expected number of matches for a search string inside the HTML panel.
