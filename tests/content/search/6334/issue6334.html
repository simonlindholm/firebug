Issue 6334: Pressing the search field shortcut should select the previously entered search text

Steps to reproduce

  1. Open Firebug on this page
  2. Focus Firebug
  3. Press Ctrl+F => The search field should be focused.
  4. Type "test" (without quotes) into the field
  5. Press Ctrl+F again => The search field value "test" should be selected
  6. Click into the main panel
  7. Press Ctrl+F again => The search field value "test" should be selected

Expected result

  • The search field value should be selected every time we click on Ctrl+F
