Command Line API: $x()

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Focus the Command Line
  4. Type $x("//button") and hit Enter
    ⇒ The console should log [button#btn1, button#btn2, button#btn3, button#btn4, button#btn5]
  5. Type $x("count(//button)") and hit Enter
    ⇒ The console should log 5
  6. Type $x("string(//button/text())") and hit Enter
    ⇒ The console should log "Button 1"
  7. Type $x("count(//button)>2") and hit Enter
    ⇒ The console should log true
  8. Type $x(".//button", document.getElementById("buttonGroup2")) and hit Enter
    ⇒ The console should log [button#btn4, button#btn5]
  9. Type $x("//button", document, "node") and hit Enter
    ⇒ The console should log <button id="btn1">
  10. Type $x("count(//button)>2", document, "number") and hit Enter
    ⇒ The console should log 1
  11. Type $x("count(//button)>2", document, "string") and hit Enter
    ⇒ The console should log "true"
  12. Type $x("//button", document, "bool") and hit Enter
    ⇒ The console should log true
  13. Type $x("test test") and hit Enter
    ⇒ The console should log an exception

Expected result

  • $x() should always return a useful result by evaluating the XPath expressions.
