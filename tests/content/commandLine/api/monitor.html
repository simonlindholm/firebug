Command line API: monitor

This test-case is intended for command line monitor method.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console and Script panel.
  2. Select the Console.
  3. Execute monitor(onExecuteTest) expression on the command line.
  4. Press this button
  5. The Console should display a log, saying that onExecuteTest has been executed + stack trace.
  6. Execute unmonitor(onExecuteTest) expression on the command line.
  7. Press this button
  8. The monitor log should not be displayed
