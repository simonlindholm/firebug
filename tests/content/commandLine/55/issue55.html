Issue 55: Auto-completion for Command Editor

Steps to reproduce

  1. Open Firebug.
  2. Enable and switch to the the Console panel.
  3. Type: function fun(anArgument1) { function fun2(anArgument2) { } anArg into the Command Editor.
  4. Press tab -> it should auto-complete to "anArgument1".
  5. Clear the editor, and type myobj.prop and press tab -> an auto-completion popup should be displayed.
  6. Press down arrow and then tab -> it should auto-complete to "prop2".
  7. Clear the editor, type document.getelementbyi and put focus after "get".
  8. Press tab -> it should auto-complete to "getElementById".
  9. Clear the editor, type document.gete and put focus at the beginning of line.
  10. Press tab -> a tab should be entered.
  11. Clear the editor, type document.get and select the last character.
  12. Press tab -> the line should be indented with no change to the selection.
