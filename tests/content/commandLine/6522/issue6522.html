Issue 6522: Implement the getEventListeners command line API

Steps to reproduce

  1. Open Firebug
  2. Enable the Console and Script panels
  3. Switch to the Console panel
  4. Verify that the following command line expressions give the correct output:
    • getEventListeners(window).load[0].useCapture -> false
    • getEventListeners(window).load[1].useCapture -> true
    • getEventListeners(document.body).click[0].listener() -> 0
    • getEventListeners(document.body).what.length -> 1
    • getEventListeners(document.body).what[0].listener(1) -> 2
