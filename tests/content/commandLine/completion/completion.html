Command Line: auto-completion

Issue 2934: 1.6a7 console and auto-complete (needs "window." prefix)

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Type doc into the Command Line and then press Tab
    ⇒ The Command Line should auto-complete the text to document.
  4. Type . into the Command Line and then press Backspace
    ⇒ The Command Line should contain document.
  5. Press Enter
    ⇒ The Command Line should be empty.
  6. Press ↑
    ⇒ The Command Line should contain document again.
  7. Press ↓
    ⇒ The Command Line should be empty again.
  8. Press ↑
    ⇒ The Command Line should contain document again.
  9. Press Escape
    ⇒ The Command Line should be empty again.
  10. Type document.id. into the Command Line and then press Tab
    ⇒ The Command Line should contain document.id..
  11. Press Enter

Expected result

  • Basic Command Line auto-completion features should work.

Issue 3600: Commandline completion rotates suggestions unnecessarily

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Execute aaaaaBBB = 1; aaaaaCCC = 2; aaaaaD = 3; in the Command Line.
  4. Type aa into the Command Line and then press Tab
    ⇒ The Command Line should contain aaaD.
  5. Press Escape
    ⇒ The Command Line should contain aa with aaaD as completion suggestion.
  6. Press Escape again
    ⇒ The Completion List Popup should be closed
    ⇒ The Command Line should still contain aa with aaaD as completion suggestion.
  7. Press Escape again
    ⇒ The Command Line should still contain aa, but there shouldn't be a completion suggestion.
  8. Type aa into the Command Line
    ⇒ The Command Line should not have any completion suggestion.
  9. Press Escape again
    ⇒ The Command Line should be empty again.
  10. Type aa into the Command Line and then press ↑ followed by Tab
    ⇒ The Command Line should contain aaaaaBBB.
  11. Press Enter
  12. Type aa into the Command Line and then press ↓ followed by Tab
    ⇒ The Command Line should contain aaaaaBBB.
  13. Press ↓
    ⇒ The Command Line should be empty again.
  14. Type aa into the Command Line and then press → followed by Tab
    ⇒ The Command Line should contain aaaaaD.
  15. Press Enter

Expected result

  • None of these should auto-complete.
