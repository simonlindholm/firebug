Copy as cURL: test multipart/form-data (issue 6817)

Test that the "Copy as cURL" menu item in the Net panel generates the proper cURL command string for multipart/form-data POST requests
Thomas Andersen, thomas@mr-andersen.no

  1. Open Firebug and select the Net panel
  2. Select a file in the file input on this page
  3. Click submit
  4. Right click on the new entry in the Net panel and execute Copy as cURL

The clipboard data should contain a curl command with the -data-binary argument. The argument should contain the multipart/form-data payload without binary data. It should be possible to use the command together with the uploaded file to simulate a file upload.
