Issue 6985: Correctly categorize DOM properties and methods

Steps to reproduce

  1. Open Firebug
  2. Turn on the Show User-defined {Properties,Functions} options in the DOM panel, and turn everything else off.
  3. Enable and switch to the Console panel
  4. Run the following expressions in the Command Line and verify that the opened DOM panel views contain only a "custom_prop" property, and __proto__:
    • inspect(window, "dom")
    • inspect(document, "dom")
    • inspect(document.body, "dom")
    • inspect(history, "dom")

Expected result

  • When "Show DOM {Properties,Functions,Constants}" are off, the DOM panel should be empty except for used-defined properties (and __proto__).
