This test-case is intended for Issue #2558 - Function arguments not expandable in watch panel even though the object is array-like.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug
  2. Enable the Script panel
  3. Press Execute Test button.
  4. Firebug should stop at debugger keyword.
  5. Type arguments in to the Watch panel and press enter.
  6. If you expand the value, you should see:
    1, "omni", 3
