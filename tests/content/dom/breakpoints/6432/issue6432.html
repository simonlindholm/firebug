Issue 6432: Long string properties don't allow to set a breakpoint for them

Steps to reproduce

  1. Open Firebug
  2. Enable the Script panel
  3. Switch to the DOM panel
    ⇒ DOM panel should list a testString property with a cropped string value.
  4. Click the breakpoint column besides that property
    ⇒ A breakpoint should be set for the property.
  5. Click the Change property value button above
    Firebug should automatically switch to the Script panel and the debugger should stop at line 16 (testString = "Now it's not a long string anymore.";)

Expected result

  • Setting breakpoints at long string properties should work.
