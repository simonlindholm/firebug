Issue 5033: Return value from a function executed on the command line is not displayed.

Steps to reproduce

  1. Open Firebug
  2. Enable the Console and Script panel and switch to the Script panel
  3. Create a breakpoint on line 10 (var a = 10;)
  4. Switch to the Console panel
  5. Type testFunction() into the Command Line and hit Enter
    The script execution should stop at the breakpoint
  6. Click the Step Over button until the execution is resumed
  7. Switch back to the Console panel and check its output

Expected result

  • The function's return value 20 should be displayed
