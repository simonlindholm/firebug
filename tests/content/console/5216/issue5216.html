Issue 5216: "Break On This Error" missing from the context menu

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Click the Cause exception button above
    ⇒ An error should be listed within the Console panel.
  4. Right-click the error
    ⇒ There should be an option called Break on This Error within the context menu.

Expected result

  • The context menu for an error message should contain an option for adding an error breakpoint for it.
