Issue 2154: Make the console API available in web workers

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Run the following commands either within w.postMessage("...") in the command line, or by typing them into the input box above and pressing return:
    • console.log('hi')
    • console.log('HI %o', {a: 1})
    • console.log('HI %c there', 'color:red')
    • console.warn('hi')
    • console.info('hi')
    • console.debug('hi')
    • console.dir({a:123})
    • console.time('a'); setTimeout(console.timeEnd.bind(console, 'a'), 100)
    • console.group('abc'); console.groupEnd()
    • console.groupCollapsed(123); console.groupEnd()
    • console.count()
    • console.assert(true, 'fail'); console.assert(false)
    • console.error('what')
    • console.exception(new Error('what'))

Expected result

  • The console API should behave roughly the same in web workers as on normal web pages.
