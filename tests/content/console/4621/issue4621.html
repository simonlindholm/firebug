Issue 4621: Multiple filters selectable in Console panel

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Reload the page
    ⇒ There should be four log messages inside the console, one normal log message, one info message, one warning and one error.
  4. Click the Info filter button
  5. Hold down Ctrl and click the Warnings filter button

Expected result

  • ⇒ The Info and the Warnings filter button should be pressed
  • ⇒ Only the info and the warning should be displayed inside the console.
