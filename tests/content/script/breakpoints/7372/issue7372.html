Issue 7372: Turning off "Log Calls" for a line hides breakpoints on that line

Steps to reproduce

  1. Open Firebug and enable the Script and Console panel
  2. Go to the Script panel, open Command Line Popup and execute step by step
    (you can also watch the Breakpoints side panel):
    debug(executeTest)
    monitor(executeTest)
    unmonitor(executeTest)
  3. The breakpoint for executeTest should be still displayed in the Script panel, but it is not -> BUG.
