Issue 7300: Watch expressions are cropped when editing them

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel and there to the Watch side panel
  3. Add a new watch expression called veryLongVariableNameToTestNameCropping
    ⇒ The name of the variable should be cropped to veryLongVa...ameCropping
  4. Right-click the watch expression and choose Edit Watch... from the context menu

Expected result

  • The editor should open and display the full name of the variable.
